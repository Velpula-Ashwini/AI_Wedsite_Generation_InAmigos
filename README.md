# 🌐 Learnify — AI-Powered Education Platform

A complete, multi-section education platform website built using AI tools as part of **Task 3** of the **InAmigos Foundation Internship Program**.


## 📌 About the Project

**Learnify** is a modern, minimal education platform website designed to showcase AI-assisted web development. The website covers multiple sections including courses, mentors, testimonials, and more — built with clean HTML and CSS.

---

## 🗂️ Sections Included

- **Navbar** — Sticky navigation with smooth scrolling
- **Hero** — Headline, CTA buttons, and key stats
- **Courses** — 4 course cards (Web Dev, Data Science, UI/UX, AI & ML)
- **About** — Platform features and step-by-step learning journey
- **Mentors** — Expert mentor profiles with ratings
- **Testimonials** — Student success stories
- **Contact / CTA** — Email signup and call-to-action
- **Footer** — Links and branding

---

## 🛠️ Tech Stack

| Technology | Usage |
|---|---|
| HTML5 | Structure and content |
